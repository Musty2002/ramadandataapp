Authentication


How to Generate (TOKEN)


The Basic Authentication Is Use To Generate (TOKEN)
It should be passed as a concatenated string like this username:password
Please use the following details for authentication
Username: Your ALBARKASUB Username
Password: Your ALBARKASUB Password
convert it to base64(username:password) use it to gerenerate token


Enpoint URL


https://albarkasub.com/api/user

Code Sample


                 $ch = curl_init();
                curl_setopt($ch, CURLOPT_URL, "https://albarkasub.com/api/user");
                curl_setopt($ch, CURLOPT_POST, 1);
                curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
                curl_setopt(
                   $ch, CURLOPT_HTTPHEADER, [
                        "Authorization: Basic ".base64_encode('username:password'),
                    ]
                );
                $json = curl_exec($ch);
                curl_close($ch);
